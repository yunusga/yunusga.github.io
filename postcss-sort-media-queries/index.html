Combine and sort CSS media queries online

A PostCSS plugin that combines and sorts CSS media queries.
It supports both mobile-first and desktop-first strategies, helping keep styles organized, predictable, and optimized.

Media query sorting options

PostCSS: 8.5.8, postcss-sort-media-queries: 6.3.2

Features

  • Merge identical CSS media queries
  • Support nested media queries
  • Support Media Queries Level 4
  • Sort media queries using mobile-first or desktop-first ordering
  • Reduce duplicated media queries
  • Keep responsive styles predictable
  • Works with any PostCSS workflow

Installation

npm install sort-css-media-queries
